# README.md

## App Name : event-balance

## Default Settings

- Language : React Native
- Build Tool : Expo
- DB : Expo SQLite (Local Storage)
- Template : blank-typescript

## App Information

- App Description : An app developed to conveniently manage family and ceremonial events on mobile devices.
- App Information : No login required

## Installation

### 프로젝트 생성 (새 프로젝트 시작 시)

```bash
npx create-expo-app@latest my-app --template blank-typescript
```

### 의존성 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npx expo start
```

### 플랫폼별 실행

```bash
# iOS 시뮬레이터
npx expo start --ios

# Android 에뮬레이터
npx expo start --android

# 웹 브라우저
npx expo start --web
```

### Expo Go 앱 사용

- iOS: App Store에서 "Expo Go" 설치
- Android: Google Play에서 "Expo Go" 설치
- QR 코드를 스캔하여 앱 실행

### SQLite 설치 (Local Storage)

```bash
npx expo install expo-sqlite
```
